# Sales-Dashboard

🎯 Sales Analytics Dashboard: Performance & Profitability Insights (Tableau)

This project is an interactive dashboard built using Tableau to analyze key business metrics across sales performance.

The dashboard focuses on core KPIs such as Total Sales, Total Profit, and Total Quantity, along with year-over-year (YoY) growth to measure business progress and performance trends.

It enables users to compare sales vs profit across categories, identify profit vs loss-making segments, and track weekly trends to understand fluctuations and seasonality. The dashboard also highlights highest and lowest performing months to quickly spot peaks and dips in performance.

The output is a set of clear, actionable insights—such as identifying high-revenue but low-profit categories, understanding regional performance variations, and detecting sales patterns over time.

Overall, the dashboard simplifies complex data into intuitive visuals, helping stakeholders make faster, data-driven decisions and improve overall business performance.
